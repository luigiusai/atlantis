Tale ricchezza sottomarina è oggi oggettivamente confermata dall'industria del Deep-Sea Mining (es. Progetto europeo *MINDeSEA*).

5. **Archeogenetica (Frozen Archive):** L'aplogruppo I-M26 e la discontinuità genetica del Mesolitico dimostrano l'isolamento dei sopravvissuti atlantidei fuggiti sulle alture (l'attuale Sardegna e Corsica).
